This is a personal fork based on oh-my-zsh.

#Install

##Linux

```bash
$ curl -L https://raw.github.com/tallmad/oh-my-zsh/master/tools/install.sh | sh
$ chsh -s /bin/zsh
```

##Mac
```bash
$ brew install zsh
$ sudo sh -c "echo '/usr/local/bin/zsh' >> /etc/shells"
$ curl -L https://raw.github.com/tallmad/oh-my-zsh/master/tools/install.sh | sh
$ chsh -s /usr/local/bin/zsh
```
